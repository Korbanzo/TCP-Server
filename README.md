# TCP-Server
This is my server-side code for a TCP connection.

# How It Works
Written using Windows Subsystem for Linux (WSL) Ubuntu
Running the file shown here starts a server on your local machine to port 54000.
From a different terminal, running the command nc localhost 54000 will connect you to the server as the client, and send a message from the server to the client.
